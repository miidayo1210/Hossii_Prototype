## 2. 解決したい課題・背景

- インキュベーション施設の会員（起業家・クリエイター）は個人作業が多く、お互いの活動が見えにくい
- 「同じ場所にいるのに、何をやっているか知らない」状態が続いている
- 施設への愛着・帰属意識が生まれにくい
- 会員同士のコラボレーション・偶発的なつながりが生まれにくい

---

## 3. 体験ゴール

### 参加者（会員）にとって

- 今日の気づき・作業中のアイデアをさっと置ける
- 他の会員の活動・気持ちが見えて「同じ場にいる感覚」が生まれる
- 施設にいることで、スペースという「共同の庭」を育てている感覚
- 過去の会員の声・アイデアとセレンディピティが起きる

### クライアント・主催者（施設運営）にとって

- 施設内の空気・雰囲気が可視化される
- 会員の活動状況・熱量が把握できる
- 施設への愛着・長期利用につながるコンテンツが蓄積される

### Hossiiとして検証したいこと

- インキュベーション施設という「アイデアが生まれる場所」でセレンディピティが起きるか
- クリエイター・起業家という独自性が高いユーザー層での利用スタイル
- 会員が入れ替わっても「場の記憶」が継承されるか

## 4. ベーススペースからの差分

> ベーススペース: [plaza-space.md](../spaces/plaza-space.md)

### 追加・変更する機能

- **アイデアタグ**: 「今日のアイデア」「閃き」などのカテゴリ投稿を推奨
- **プロジェクトタグ（将来）**: 自分のプロジェクト名タグを付けて投稿できる
- **会員同士のゆるい連携**: 誰かのアイデアに「+1」できる簡易いいね機能（将来）
- **新会員歓迎モード（将来）**: 新しい会員が入ったとき、過去の会員の声でお迎えする

### 筑波山スペースとの比較

| 項目 | 筑波山 | Inspired Lab |
|------|--------|--------------|
| 来訪者 | 通りすがり・観光客 | 継続利用の会員 |
| 投稿の性質 | 訪問の記念・感動 | 日常の気づき・アイデア |
| 関係性の深さ | 浅い（一期一会） | 深い（継続的なつながり） |
| セレンディピティの方向 | 過去の来訪者との出会い | 仲間のアイデアとの化学反応 |

### Feature Flag 設定

| Flag Key | ON/OFF | 理由 |
|----------|--------|------|
| `commentPost` | ON | アイデア・気づき |
| `emotionPost` | ON | 今日の気持ち |
| `photoPost` | ON | 作業中の画面・プロトタイプ写真 |
| `numberPost` | OFF | 不要 |
| `random_recall_enabled` | ON | 過去会員のアイデアとの出会い |
| `public_board_mode` | ON（検討） | 施設の外からも見えると施設の魅力発信になる |
| `comments_thumbnail` | ON | 写真表示 |

### カスタム感情セット

- `inspire`（アイデア）・`wow`（すごい）・`think`（考え中）・`fun`（楽しい）・`joy`（うれしい）
- インキュベーション施設らしい「創造・挑戦」の文脈に合わせる

## 7. 成功指標（KPI）

| 指標 | 目標 | 計測方法 |
|------|------|---------|
| 会員参加率 | 会員の60%以上が1回以上投稿 | authorId / 会員数 |
| 月間アクティブ投稿者 | 会員の30%以上が月1投稿以上 | 月次投稿者数 |
| 写真・アイデア投稿率 | 全投稿の20%以上 | imageUrl あり比率 |
| 施設への愛着スコア | 「Inspired Labにいたくなる」の肯定率向上 | 定期アンケート |
