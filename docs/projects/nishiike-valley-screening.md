## 2. 解決したい課題・背景

- 映画鑑賞後の「感想を言いたいけど言えない」という場の空気
- 全員が発言できるわけではなく、感想が少数の声に偏る
- アンケート（紙・フォーム）は後で集めても会場の熱量が失われる
- 鑑賞中・直後のリアルな感情をリアルタイムで可視化したい

---

## 3. 体験ゴール

### 参加者にとって

- 映画を観ながら・観た直後に、リアルな感情を匿名で置ける
- 他の人の感想が見えて「同じように感じた人がいる」と安心・共感できる
- 一言コメントや感情タグで「発言」のハードルが下がる

### クライアント・主催者にとって

- 参加者の感想・感情が可視化され、次回企画に活用できる
- 会場のスクリーンに投稿が流れることで、鑑賞後の対話が生まれる
- アンケート不要・または補完できる

### Hossiiとして検証したいこと

- 映画鑑賞という「感情が高まる体験」直後に投稿が増えるか
- 匿名投稿で普段言えない感想が出てくるか

## 4. ベーススペースからの差分

> ベーススペース: [workshop-space.md](../spaces/workshop-space.md)

### 追加・変更する機能

- **リアクション重視の感情セット**: 映画特有の感情に合わせたカスタマイズ
- **鑑賞中投稿モード**: 映画を観ながら静かに投稿できるUI（暗い画面対応・バイブのみ）
- **ネタバレ注意フラグ**: コメントに「ネタバレあり」タグを付けられると親切

### 使わない機能

- ZINE出力: 不要（感想まとめは別途対応）
- セッション管理: 映画1本 = 1スペースで十分
- 画像投稿: 映画中は不要（終了後の写真は可）

### Feature Flag 設定

| Flag Key | ON/OFF | 理由 |
|----------|--------|------|
| `commentPost` | ON | 感想・一言 |
| `emotionPost` | ON | 鑑賞中のリアクション |
| `photoPost` | OFF（または終映後ON） | 映画中は不要 |
| `numberPost` | OFF | 不要 |
| `zine_export_enabled` | OFF | 不要 |
| `comments_thumbnail` | ON | 終映後に写真があれば表示 |
| `public_board_mode` | OFF | 参加者限定 |

### カスタム感情セット

- `moved`（ぐっときた）・`wow`（すごい）・`think`（考えた）・`laugh`（笑った）・`fun`（楽しい）・`inspire`（閃いた）
- 映画・芸術鑑賞に合わせた語感に変更を検討（「ぐっときた」→「泣いた」など）

## 6. 運用フロー

### 事前準備

- [ ] 映画ごとにスペースを作成（または1スペースを流用）
- [ ] QRコードをスクリーン投影 or 紙で配布
- [ ] 鑑賞後に全投稿をスクリーンに表示（スペースビュー投影）

### 当日 / 運用中

- 鑑賞開始時にQRを案内
- 鑑賞中・終映直後に自由投稿
- 終映後、投稿をスクリーンに投影して対話の起点にする

### 終了後・アーカイブ

- 感想の一覧をエクスポートして主催者に共有
- 次回の企画・映画選定に活用
