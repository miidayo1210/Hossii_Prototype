## 1. 概要

| 項目 | 内容 |
|------|------|
| スペース名 | 内省スペース |
| 一言説明 | 自分の気持ち・アイデア・気づきを置いておくと、後で偶然また出会える個人の庭 |
| 目的 | 整理しなくていい。置くだけで、過去の自分・アイデアとセレンディピティが起きる |
| カテゴリ | 個人系 |

---

## 2. ユーザー

| 役割 | 説明 |
|------|------|
| 主ユーザー（本人） | 自分のスペースを持ち、日常的に気持ち・アイデアを投稿する |
| 副ユーザー | 基本なし（プライベート運用が基本）/ 任意で他者に公開可 |
| 運営・管理者 | 本人が管理者を兼ねる |

---

## 3. 利用シーン

| 項目 | 内容 |
|------|------|
| いつ | 日常の思いついた瞬間、振り返り時、就寝前、通勤中など |
| どこで | スマートフォンからどこでも |
| 何人規模 | 1人（個人）|
| 利用デバイス | スマートフォンメイン |
| 頻度 | 1日数回〜週数回 |

---

## 4. 体験価値

### 本人に何が起きるか

- 「整理しなくていい」安心感で、思考の断片を気軽に置ける
- 忘れていたアイデア・気持ちが予期せず現れ、新しい文脈で繋がる
- 時系列で自分の変化・成長が見える
- 過去の自分が「励まし」や「ヒント」になる

### Hossiiとして何を検証したいか

- 個人利用での継続率（毎日置き続けるか）
- セレンディピティ体験の発生率・満足度
- 「整理不要」の設計で情報量が増えるか

| 入力種別 | 説明 | 必須/任意 |
|---------|------|----------|
| 気持ちタグ | 今の感情 | 任意 |

|------|------|

| 匿名/記名 | 記名（自分） |

## 6. 表現設計（Render）

### メイン画面での表示

- バブル型でふわふわと自分の投稿が漂う
- 感情別に色が変わり、「今の自分の庭」の景色になる

### セレンディピティ表示

- **ランダム想起**: 過去の投稿が不意に浮かび上がる（「3ヶ月前のあなたは…」）
- **今日の過去**: 同じ日の過去年の投稿が現れる（「去年の今日、こんな気持ちだったよ」）
- **似た気持ち**: 同じ感情タグの過去投稿をサジェスト

### タイムライン・振り返り

- カレンダー型で感情の変化を振り返る
- 気持ち別でフィルタリング
- 「アイデア」タグで絞り込み

### 蓄積・成果物（Archive）

- 月次の感情サマリー
- 「今月の3大気づき」自動抽出（将来）
- PDF出力（振り返りレポート）

## 8. データ仕様

### Coreからの追加フィールド（候補）

| フィールド | 型 | 説明 | 必須 |
|-----------|-----|------|------|
| `tags` | string[] | ユーザー自由タグ（「アイデア」「仕事」「家族」など） | 任意 |
| `mood` | number (1-5) | 気分スコア（数値投稿の特殊版） | 任意 |
| `isPrivate` | boolean | 公開/非公開フラグ | デフォルトtrue |

### スペース設定

| 設定キー | 推奨値 | 説明 |
|---------|--------|------|
| `quickEmotions` | 全8種 | 細かい感情を捉えるため全部使う |
| `cardType` | `graph` | 感情の変化をグラフで見せる |

## 12. Coreとの差分

### 新しく必要な機能

- **ランダム想起エンジン**: 過去投稿をランダム・文脈的にサジェストするロジック
- **内省ビュー** (`#reflection`): カレンダー×感情グラフの専用画面
- **プライベートスペース**: URLを知っていても他者から見えないスペース
- **自由タグ**: 感情タグとは別にユーザーが任意のタグを付けられる

### 不要なCore機能

- ゲスト入室フロー（個人スペースなので不要）
- 公開スクリーン投影

### Hossiiとの哲学的な整合

- このスペースはセレンディピティ設計の最も純粋な実装形
- 「整理させない」「出会わせる」の体験が最も直接的に発揮される

## 13. 空間状態の遷移

| フェーズ | 状態 | 表現 |
|---------|------|------|
| 使い始め（0〜7日） | 投稿が少なく庭が静か | バブル数個。「最初の気持ちを置いてね」 |
| 習慣化（1〜4週） | 毎日少しずつ積み重なる | バブルが増え、色のグラデーションが生まれる |
| セレンディピティ期（1ヶ月〜） | 過去の自分が現れ始める | ランダム想起が発動。「懐かしい気持ち」が蘇る |
| 成熟期（3ヶ月〜） | 自分の庭が独自のエコシステムを持つ | アイデアが連鎖し、感情の変化が可視化される |
