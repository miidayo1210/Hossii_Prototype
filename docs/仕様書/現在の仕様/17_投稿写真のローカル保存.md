Hossiiの投稿画面で、写真を投稿する際に撮影した画像は、ローカル端末に保存されるようにする
Hossiiのお絵描きの投稿画面でも、指で書いたイラストは、ローカル端末に保存されるようにする

---

## 実装方針

| 対象 | 実装 |
|------|------|
| カメラ撮影（📷） | `capture="environment"` の専用 input を用意し、撮影後に `saveImageLocally()` を呼ぶ |
| ギャラリー選択（📸） | すでにデバイスにあるためローカル保存処理は不要 |
| お絵描き（DrawingModal） | Canvas → Blob → File に変換後、`saveImageLocally()` を呼んでから投稿フローへ |

## 保存ロジック（`saveImageLocally.ts`）

1. `navigator.canShare({ files })` が使える場合（iOS Safari / PWA）→ Web Share API でシェートシート経由でデバイス保存
2. 非対応環境 → `<a download>` でファイルダウンロード
3. ユーザーがキャンセルした場合や失敗した場合でも投稿フローは止めない
