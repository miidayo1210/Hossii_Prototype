## 概要

スペース画面の左コントロールバーにある「表示」セクションに
スライドショーモードを追加する。
iPad や iPhone などの小さな画面で投影する際に、
投稿を1枚ずつ大きく・自動でめくれるように表示する機能。

---

## UI 配置

### LeftControlBar の表示ゾーン

現在の VIEW_MODE_OPTIONS に slideshow を追加する。

| ボタン | アイコン | 説明 |
|--------|---------|------|
| フル表示 | 📝 | 既存 |
| バブル表示 | 💬 | 既存 |
| 画像のみ | 🖼 | 既存 |
| スライドショー | 🎞 | 新規追加 |

## スライドショー画面の仕様

### 表示レイアウト

- 画面全体を使って投稿を1枚ずつ表示する
- 左コントロールバー・トップバー・ボトムナビはすべて非表示にする
  （フルスクリーンに近い状態で投影向けの見た目にする）
- 投稿の表示形式は「フル表示」と同等
  （感情アイコン・メッセージ・画像をすべて表示）
- 背景はスペースの背景設定をそのまま引き継ぐ

### スライドの並び順

- デフォルトはランダム順
- 表示件数・期間フィルタは LeftControlBar で設定した値をそのまま引き継ぐ

### 自動送り

- 設定した秒数ごとに次の投稿へ自動で切り替わる
- デフォルト: 5秒
- 設定可能な間隔: 3秒 / 5秒 / 10秒 / 20秒
- スライド切り替えはフェードトランジション

### 手動送り

- 画面の右半分をタップ → 次の投稿へ
- 画面の左半分をタップ → 前の投稿へ
- キーボードの → キー・← キーでも操作可能（PC 投影向け）

### 最後のスライドに達したとき

- 先頭に戻ってループ再生する

### 設定パネル

- スライドショー表示中に画面下部から設定バーを表示する
- 操作がない場合、3秒後に設定バーは自動で非表示になる
- 画面をタップすると設定バーが再表示される
- 設定項目:
  - 自動送りの間隔（3 / 5 / 10 / 20 秒）
  - 一時停止 / 再開トグル
  - スライドショー終了ボタン（元の viewMode に戻る）

## 実装方針

### ViewMode への追加

`src/core/utils/displayPrefsStorage.ts` の ViewMode 型に slideshow を追加する。

```ts
export type ViewMode = 'full' | 'bubble' | 'image' | 'slideshow';
```

### SpaceScreen での分岐

SpaceScreen で viewMode === 'slideshow' のときに、
既存のバブルレイアウトの代わりに SlideshowView を描画する。
LeftControlBar・TopBar・BottomNavBar はこのモード中は非表示にする。

### SlideshowView コンポーネント（新規）

`src/components/Slideshow/SlideshowView.tsx` として作成する。

| Props | 型 | 説明 |
|-------|----|------|
| hossiis | Hossii[] | 表示対象の投稿配列（フィルタ済み） |
| onExit | () => void | スライドショー終了時のコールバック |

内部状態:
- currentIndex: 表示中のスライドインデックス
- isPlaying: 自動送り中かどうか
- interval: 自動送りの間隔（秒）
- showControls: 設定パネルの表示状態
