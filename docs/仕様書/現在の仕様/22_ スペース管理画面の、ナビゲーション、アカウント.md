# スペース管理画面 — ナビゲーション・アカウントメニュー仕様

> **分類:** `[UI]` スペース管理画面 / アカウント管理
> **関連:** `21_コミュニティごとのスペース管理画面.md`、`03_スペースURL機能.md`

> 最終更新: 2026-03-06（実装完了）

---

## 概要

スペース管理画面（`SpacesScreen`）のヘッダー右端にある「アカウント」ボタンを押すと、
ドロップダウンメニューが表示される。
現在はコミュニティ名・登録メールアドレス・ログアウトボタンが表示されるが、
**コミュニティ ID（slug）の表示と編集**を追加する。

---

## 現在の表示内容（実装済み）

| 項目 | 内容 |
|------|------|
| コミュニティ名 | `currentUser.communityName` |
| メールアドレス | `currentUser.email` |
| ログアウトボタン | セッション終了 |

---

## 追加仕様

### 1. コミュニティ ID の表示

アカウントメニューに、現在のコミュニティ slug を表示する。

```
コミュニティ ID: a3f9c2b1
```

- `currentUser.communitySlug` が `null` の場合は「未設定」と表示する。
- 表示は読み取り専用テキスト + 編集ボタン（鉛筆アイコン）の構成とする。

---

### 2. コミュニティ ID の編集

編集ボタンを押すとインライン編集モードになる。

#### 入力バリデーション（`spaceURL` と同じルール）

| ルール | 内容 |
|--------|------|
| 使用可能文字 | 半角英小文字・数字・ハイフン（`a-z 0-9 -`） |
| 文字数 | 3〜40 文字 |
| 先頭・末尾 | ハイフン不可 |
| 一意性 | 他のコミュニティと重複不可（DB `UNIQUE` 制約） |

#### 保存フロー

1. バリデーション通過後に「保存」ボタンが活性化
2. 保存実行 → `communities` テーブルの `slug` を UPDATE
3. 保存完了後、`currentUser.communitySlug` を更新（再ログイン不要）

---

### 3. URL 変更に関する注意書き

編集モード中、入力欄の下に以下の注意書きを常時表示する：

> ⚠️ コミュニティ ID を変更すると、このコミュニティの全スペースの招待 URL が変わります。
> 配布済みの QR コードや URL は無効になります。

- 保存ボタンを押す前に確認ダイアログ（`confirm`）でもう一度警告する：
  > 「コミュニティ ID を変更すると、配布済みの URL・QR コードがすべて無効になります。変更しますか？」
- キャンセルした場合は保存しない。
