# まち歩きWS用 複数画像投稿機能

## 概要

まち歩きワークショップなど、**写真記録が中心となるイベント**向けの追加機能。  
Feature Flags でオンにすることで有効化できる。

1枚の写真をその場で投稿し、ログ一覧で**複数枚の写真を時系列に並べて比較・観測**できる UI を提供する。

---

## ユースケース

- まち歩き中に撮影した写真（Before / After、複数地点など）を 1 投稿にまとめる
- ログ一覧で同じ投稿の写真を横に並べて時系列・視点の変化を比較する
- WS 参加者がその場でスマホで投稿→ファシリテーターが大画面で並べて振り返る

---

## Feature Flag

| フラグキー | デフォルト | 説明 |
|-----------|-----------|------|
| `multi_photo_post` | `false` | 複数画像投稿 UI を有効化（投稿フォームと一覧 UI の両方に影響） |

有効化はスペース設定 → Feature Flags タブから行う。

---

## 仕様詳細

### 投稿フォーム（PostScreen）

- **最大 4 枚**まで画像を選択できる
- `<input type="file" accept="image/*" multiple>` で複数選択
- 選択した画像のプレビューを横並びサムネイルで表示
- 各プレビューに **×（削除）ボタン** を配置
- 順序の変更は不要（選択順で保存）
- フラグ OFF の場合は従来通り 1 枚のみ選択可能

#### バリデーション

| 条件 | 制限 |
|------|------|
| 枚数上限 | 4 枚（5 枚目を追加しようとしたらトースト表示） |
| ファイルサイズ | 既存の圧縮処理（最大 1280px / 1MB）を各ファイルに適用 |
| ファイル形式 | `image/*`（既存と同じ） |

---

### データモデル

#### 型定義（`src/core/types/index.ts`）

現状 `imageUrl?: string`（単数）に加え、`imageUrls?: string[]` を追加する。  
後方互換のため `imageUrl` は残す。

```ts
// 既存（後方互換のため残す）
imageUrl?: string;
// 追加
imageUrls?: string[];  // 複数画像URL（最大4件）
```

`AddHossiiInput` にも同様に `imageUrls?: string[]` を追加。

#### Supabase テーブル（`hossiis`）

`image_urls` カラム（`text[]` 型）を追加するマイグレーションが必要。

```sql
ALTER TABLE hossiis ADD COLUMN image_urls text[] DEFAULT '{}';
```

#### ストレージ（`hossii-images` バケット）

現状: `{spaceId}/{hossiiId}.jpg`（1枚）  
変更後: 複数枚は連番インデックスで管理

```
{spaceId}/{hossiiId}_0.jpg
{spaceId}/{hossiiId}_1.jpg
{spaceId}/{hossiiId}_2.jpg
{spaceId}/{hossiiId}_3.jpg
```

---

### ログ一覧（CommentsScreen）

`imageUrls` が存在する場合、**横並びグリッド**で表示する。

#### 表示ルール

| 枚数 | レイアウト |
|------|-----------|
| 1 枚 | 現状と同じ（サムネイル 1 枚） |
| 2 枚 | 横 2 列グリッド（各 50%） |
| 3 枚 | 横 3 列グリッド（各 33%）または 2+1 レイアウト |
| 4 枚 | 横 2×2 グリッド |

- 各サムネイルのアスペクト比は **1:1**（`object-fit: cover`）で統一し、高さを揃えることで比較しやすくする
- サムネイルをタップ → 既存のライトボックスで 1 枚ずつ拡大表示（左右スワイプ or ← → ボタンで切り替え）

#### Feature Flag との関係

| `multi_photo_post` | `comments_thumbnail` | 表示 |
|--------------------|--------------------|------|
| OFF | ON | 従来通り（単数サムネイル） |
| ON | ON | 複数サムネイルグリッド |
| — | OFF | `📎 画像を開く` リンク（変更なし） |

## 実装が必要なファイル一覧

| ファイル | 変更内容 |
|---------|---------|
| `src/core/types/index.ts` | `Hossii` と `AddHossiiInput` に `imageUrls?: string[]` を追加 |

| `src/core/utils/imageStorageApi.ts` | `uploadHossiiImages(spaceId, hossiiId, files[])` を追加（連番パス対応） |
| `src/core/utils/hossiisApi.ts` | Supabase 保存・取得時に `image_urls` ↔ `imageUrls` のマッピングを追加 |

| Supabase マイグレーション | `hossiis` テーブルに `image_urls text[]` カラム追加 |
