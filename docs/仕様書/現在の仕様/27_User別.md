# ユーザーロール別 機能・権限まとめ

## ロールの種類

| ロール | 識別フラグ | Supabase `app_metadata.role` |
|--------|-----------|------------------------------|
| **スーパー管理者** | `isSuperAdmin: true` | `"super_admin"` |
| **管理者**（コミュニティ） | `isAdmin: true`, `isSuperAdmin: false` | `"admin"` |
| **一般ユーザー / ゲスト** | `isAdmin: false` | なし |

ロール解決は `src/core/contexts/AuthContext.tsx` の `resolveAppUser()` で行われる。

---

## スーパー管理者

### ログイン後の遷移
- `#communities`（コミュニティ一覧画面）へ自動遷移

### できること
- **全コミュニティの閲覧**（Supabase RLS で `super_admin` のみ全件 SELECT 可）
- **各コミュニティのステータス確認**（`approved` / `pending` / `rejected`）
- **任意コミュニティのスペース管理画面へ遷移**（`overrideCommunityId` 経由）
- スペース管理画面で「**戻る**」ボタンからコミュニティ一覧に戻れる

### できないこと（現時点の制限）
- コミュニティの承認・却下 UI（現在は Supabase Dashboard での手動 SQL）
- スペースの作成・削除の永続化（RLS 修正が必要）
- 自分の `communityId` を持たないため、コミュニティ ID セクションは非表示

---

## 管理者（コミュニティ管理者）

### ログイン後の遷移
- `#spaces`（自コミュニティのスペース一覧）へ自動遷移

### コミュニティ登録ステータスによる制御

| `communities.status` | ふるまい |
|---------------------|---------|
| `approved` | `isAdmin: true` → SpacesScreen へ遷移 |
| `pending` | `isAdmin: false` → 「審査中」画面を表示 |
| `rejected` | `isAdmin: false` → 「申請が承認されませんでした」＋自動ログアウト |
| 未登録 | エラーメッセージ表示・自動ログアウト |

### スペース管理（SpacesScreen）

| 機能 | 説明 |
|------|------|
| スペースの**作成** | 新規作成モーダルから作成 |
| スペースの**名前変更** | インライン編集 |
| スペースの**ID（slug）変更** | インライン編集 |
| スペースの**削除** | 確認ダイアログ付き |
| スペースの**背景変更** | |
| 招待 URL の**コピー** | |

### スペース設定（SpaceSettingsScreen）— 管理者のみ表示されるタブ

| タブ | 説明 |
|------|------|
| **タグ設定** | スペースのプリセットタグ管理 |
| **モデレーション** | 投稿の非表示・復元 |
| **Feature Flags** | スペース機能フラグの ON/OFF |

> 一般ユーザーには上記3タブは表示されない。

管理者・一般ユーザー共通で表示されるタブ：基本設定、Hossii カスタマイズ、背景設定、シェア/QR、スペース装飾

### 投稿の非表示・復元（モデレーション）

- スペース画面の吹き出し選択時に「🚫 非表示」ボタンが表示される（管理者のみ）
- モデレーションタブからスペース内全投稿（非表示含む）を一覧・操作できる
- 非表示にした投稿は `hossiis.is_hidden = true` に更新され、全ユーザーから見えなくなる
- 操作は `moderation_logs` テーブルに `hide` / `restore` + `admin_id` で記録される
- ログ一覧（`#comments`）からも非表示になる（仕様: `28_スペース投稿非表示.md`）

### バブル編集権限

- 管理者は設定に関わらず**常に全バブルを編集可能**

### スペース設定からの「戻る」ボタン
- 管理者 → `#spaces`（スペース一覧）
- 一般ユーザー → `#screen`（スペース画面）

## 一般ユーザー / ゲスト

### 種別の違い

| 種別 | 認証 | 過去ログ閲覧 | オンボーディング |
|------|------|------------|----------------|
| ログイン済みユーザー | Supabase Auth（メール / Google） | 閲覧可（未実装） | ニックネーム入力 |
| ゲスト参加者 | 不要（匿名） | 閲覧不可 | ニックネーム入力のみ |

### できること

- 投稿（Hossii）の作成
- スペース画面の閲覧
- ログ一覧（`#comments`）の閲覧
- 自分のバブルの移動・リサイズ・色変更（スペース設定が「全員編集可」の場合）
- ニックネームの設定

### できないこと（管理者専用機能）

- スペースの作成・削除・名前変更・ID変更・背景変更
- 他者の投稿の非表示・復元
- モデレーションタブへのアクセス
- タグ設定タブへのアクセス
- Feature Flags タブへのアクセス
- 吹き出し選択時の「🚫 非表示」ボタン操作
