## 概要

スペース画面の吹き出しフッター帯に配置されている「いいね」ボタン（❤️）のインタラクションを気持ちよくする。

| # | 要件 | ゴール | 状態 |
|---|---|---|---|
| B | ボタン自体をぷにぷにさせたい | 押す前はふくらんでいて、押すと凹み、離すと跳ね返る | ✅ 実装済み |
| P | ハート・星をボタンから飛び出させたい | クリック時にパーティクルが舞い上がる | ✅ 実装済み |
| H | Hossii がニコッと笑ったり踊ったりしてほしい | いいね時に HossiiLive がリアクション表情を出す | ✅ 実装済み |

## B: ボタンのアニメーション

### 設計方針

- **未押下（アイドル）**: ゆっくりと微かに拍動（pulse）するアニメーションで生き物感を出す。
- **ホバー**: 少し膨らむ（`scale(1.12)`）。
- **クリック瞬間**: ぐっと凹む（`scale(0.82)`）→ 跳ね返り（`scale(1.18)`）→ 定常（`scale(1)`）。CSS の `cubic-bezier` でバネのような弾力感を表現する。
- **押下後（liked 状態）**: ハートが赤くなり、カラーが `#ef4444` に変わる。

> **パフォーマンス方針:** 多数の吹き出しが並んでも重くならないよう、pulse の scale 幅を `1.06` に抑えている（仕様原案の 1.08 より小さい）。

### B01: CSS アニメーション

```css
/* アイドル: 微かに脈打つ */
@keyframes likePulse {
  0%, 100% { transform: scale(1); }
  50%       { transform: scale(1.06); }
}

/* クリック: 凹む → 跳ね返り */
@keyframes likeBounce {
  0%   { transform: scale(1); }
  20%  { transform: scale(0.82); }
  55%  { transform: scale(1.18); }
  80%  { transform: scale(0.96); }
  100% { transform: scale(1); }
}

.likeButton {
  animation: likePulse 2.4s ease-in-out infinite;
  transition: background 0.15s, color 0.15s, border-color 0.15s;
}

.likeButton:hover {
  transform: scale(1.12);
  animation: none; /* ホバー中は pulse を止める */
  background: rgba(239, 68, 68, 0.08);
}

.likeButtonBouncing {
  animation: likeBounce 0.45s cubic-bezier(0.36, 0.07, 0.19, 0.97) forwards;
}

.likeButtonActive {
  color: #ef4444;
  border-color: rgba(239, 68, 68, 0.4);
  background: rgba(239, 68, 68, 0.08);
}
```

### B02: React 実装

`Bubble` コンポーネント内でいいね押下ごとに `likeButtonBouncing` クラスを付与し、アニメーション完了後に削除する。

```tsx
const [isBouncing, setIsBouncing] = useState(false);
const [isLiked, setIsLiked] = useState(false);

// onClick ハンドラ内
if (isBouncing) return;
setLocalLikeCount((c) => c + 1);
setIsLiked(true);
setIsBouncing(true);
setTimeout(() => setIsBouncing(false), 450); // likeBounce と同じ時間
onLike?.(hossii.id);

// JSX
<button
  className={[
    styles.likeButton,
    isLiked ? styles.likeButtonActive : '',
    isBouncing ? styles.likeButtonBouncing : '',
  ].filter(Boolean).join(' ')}
  onPointerDown={(e) => e.stopPropagation()}
  onClick={handleLike}
  aria-label="いいね"
>
  ❤️ {localLikeCount > 0 && <span>{localLikeCount}</span>}
</button>
```

## P: パーティクルエフェクト

### 設計方針

- いいねボタンを押した瞬間、ボタン付近からハート・星が飛び出して上方向に舞い上がる。
- パーティクルは `position: absolute`（ボタンを囲む `div` 内）で生成し、CSS アニメーションで上方向 + 拡散 + フェードアウトさせる。
- 個数: `3〜5` 個（ランダム）。絵文字: `❤️` / `💛` / `⭐` / `✨` をランダムで選ぶ。

> **パフォーマンス方針:** 浮上距離を `-44px`（原案 `-50px`）、フォントサイズを `13px`（原案 `14px`）、持続時間を `0.65s`（原案 `0.7s`）に抑え、視覚的なうるさみを軽減している。

### P01: パーティクル CSS

```css
@keyframes likeParticle {
  0%   { transform: translate(0, 0) scale(1); opacity: 1; }
  80%  { opacity: 0.6; }
  100% { transform: translate(var(--tx), -44px) scale(0.4); opacity: 0; }
}

.likeParticle {
  position: absolute;
  bottom: 100%;
  left: 50%;
  font-size: 13px;
  pointer-events: none;
  animation: likeParticle 0.65s ease-out forwards;
}
```

- `--tx` はインライン style で各パーティクルにランダムな横移動量（`-20px`〜`+20px`）を設定する。

### P02: React 実装

```tsx
type LikeParticle = { id: string; emoji: string; tx: number };
const [likeParticles, setLikeParticles] = useState<LikeParticle[]>([]);

// onClick ハンドラ内に追加:
const LIKE_EMOJIS = ['❤️', '💛', '⭐', '✨'];
const count = 3 + Math.floor(Math.random() * 3); // 3〜5個
const particles: LikeParticle[] = Array.from({ length: count }, (_, i) => ({
  id: `${Date.now()}-${i}`,
  emoji: LIKE_EMOJIS[Math.floor(Math.random() * LIKE_EMOJIS.length)],
  tx: Math.round((Math.random() - 0.5) * 40),
}));
setLikeParticles(particles);
setTimeout(() => setLikeParticles([]), 750); // アニメーション完了後に削除

// フッター帯内のボタンを relative div で囲む:
<div style={{ position: 'relative' }}>
  {likeParticles.map((p) => (
    <span
      key={p.id}
      className={styles.likeParticle}
      style={{ '--tx': `${p.tx}px` } as React.CSSProperties}
    >
      {p.emoji}
    </span>
  ))}
  <button ...>...</button>
</div>
```

## H: Hossii のリアクション

### 設計方針

- いいねが押された時、`SpaceScreen` 経由で HossiiLive にリアクショントリガーを送る。
- HossiiLive は受け取ると笑顔表情を一定時間表示する。
- 既存の `reactionFace` の仕組みに乗る形で、「いいね専用 useEffect」を独立して追加する（`lastTriggerId` の Layer B とは干渉しない）。

### H01: SpaceScreen への通知

```tsx
// SpaceScreen.tsx
const [likeReactionTrigger, setLikeReactionTrigger] = useState<{ id: string } | null>(null);

const handleLike = useCallback(async (hossiiId: string) => {
  try {
    await incrementLike(hossiiId);
    setLikeReactionTrigger({ id: `like-${hossiiId}-${Date.now()}` });
  } catch (err) {
    console.error('[SpaceScreen] incrementLike error:', err);
  }
}, []);

// HossiiLive への受け渡し
<HossiiLive
  ...
  onLikeTrigger={likeReactionTrigger?.id}
/>
```

### H02: HossiiLive に `onLikeTrigger` prop を追加

```ts
// Props 型
onLikeTrigger?: string | null; // trigger ID（変化があれば反応）
```

`onLikeTrigger` が変化したとき、喜び系表情（`joy` / `fun` / `laugh` からランダム）をセットし、1.2〜1.5 秒後に idle に戻す。

```ts
useEffect(() => {
  if (!onLikeTrigger) return;
  const joyFaces = ['joy', 'fun', 'laugh'] as EmotionKey[];
  const face = getHossiiFace(joyFaces[Math.floor(Math.random() * joyFaces.length)]);
  setReactionFace(face);
  const timer = setTimeout(() => setReactionFace(null), 1200 + Math.random() * 300);
  return () => clearTimeout(timer);
}, [onLikeTrigger]);
```
