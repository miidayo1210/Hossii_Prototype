## スペース管理画面（SpacesScreen）

スペース一覧の管理・編集・削除を行う管理者向け画面。

### スペースカード

各スペースカードに以下の操作が可能。

- **背景サムネイル**：クリックで背景セレクターを展開
- **スペース名**：鉛筆アイコンでインライン編集
- **カードタイプ**：ドロップダウンで切り替え（星座 / スタンプ）
- **スペースID（slug）**：鉛筆アイコンでインライン編集
- **招待URLをコピー**：ボタンクリックでクリップボードへコピー
- **設定（歯車アイコン）**：スペース設定画面へ遷移
- **削除（🗑アイコン）**：確認ダイアログ後にスペースを削除

### スペース削除の挙動

1. 確認ダイアログ（「この操作は取り消せません」）を表示
2. 確認後、ローカル状態からスペースを即時削除（楽観的更新）
3. Supabase に DELETE を非同期送信
4. **DELETE 失敗時**（RLS ブロック・通信エラー等）：
   - ローカル状態を自動ロールバック（スペースを復元）
   - アラートでユーザーにエラーを通知
   - コンソールに認証状態・エラー詳細をログ出力

> ⚠️ RLS（Row Level Security）で DELETE がブロックされた場合、スペースは画面上は一時的に消えるが即座に復元される。Supabase ダッシュボードで `spaces` テーブルの DELETE ポリシーを確認すること。
