# 39: 管理スペースからスペースの名前を変更

## 背景・課題

コミュニティのスペース管理画面では、管理するスペースの一覧が表示され、スペース名を編集できる。
しかし以下の2つのバグが存在していた。

1. **変更がリロードで元に戻る**: スペース名を変更しても、ページをリロードすると変更前の名前に戻ってしまう
2. **URLで開いたスペースに反映されない**: スペース管理画面で名前を変更しても、実際にURLで開いたスペースのタイトルには変更が反映されない

## 目標

スペース管理画面から、スペース名を永続的かつ即時に変更できるようにする。

## 修正内容

### 原因調査・修正

- スペース名の変更がデータ永続化（Supabase / ローカル状態）に正しく反映されていない箇所を特定する
- 変更をデータソースに保存し、リロード後も維持されるようにする
- URLで開いたスペース画面も同じデータを参照するよう整合させる

### UXフロー（変更後）

1. スペース管理画面でスペース名の編集フィールドを操作
2. 保存ボタン（または Enter）で確定
3. ページリロード後も変更が維持される
4. URLで直接開いたスペースのタイトルにも即時反映される

## ステータス

- 修正を実施済みの可能性あり（要動作確認）
  - リロード後もスペース名が維持されるか
  - URLで開いたスペースのタイトルに変更が反映されるか
