# 40: コミュニティ一覧から各コミュニティの詳細が少し見えるようにする

## 背景・課題

superadmin のコミュニティ一覧画面では、各コミュニティの「スペース管理」ボタンを開かないと中身が確認できない。
一覧ダッシュボードを見ただけでは、各コミュニティの状況がまったくわからない状態になっている。

## 目標

コミュニティ一覧から、各コミュニティのアクティビティ概要をひと目で把握できるようにする。

## 修正内容

### トグル UI の追加

- 各コミュニティの行に **トグルボタン** を設置する
- トグルを押すと、下方向にアコーディオン展開して詳細情報が表示される
- もう一度押すと閉じる

### 展開時に表示する情報

| 項目 | 内容 |
|------|------|
| スペース数 | そのコミュニティ内のスペース件数 |
| 最終編集日時 | スペースまたは投稿の最終更新日時 |
| 管理者 | コミュニティを管理しているユーザー名 |
| 投稿数 | これまでの総投稿数（またはアクティビティ数） |

### UXフロー

1. superadmin がコミュニティ一覧画面を表示
2. 気になるコミュニティのトグルをクリック
3. 一覧の下にビヨーンと詳細が展開される
4. 再度クリックで折りたたむ

## 備考

- スペース管理画面へ遷移しなくても概要が確認できることを優先する
- 詳細な編集・操作は引き続きスペース管理画面で行う
