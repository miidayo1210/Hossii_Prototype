# 42: スマホ参加UIの改善

## 背景・課題

スペースURLを開いたときのスマホ向け参加フローで、以下の問題がある。

1. **スペース名の二重表示**: ページ上部にスペース名が2つ表示されている
   - 1つ目: スペース名のみのタイトル（グレー）
   - 2つ目: リンクコピーボタン付きのスペース名
2. **投稿の視覚表現がわかりにくい**: 小さな粒がコメント・気持ちのかけらを表しているが、直感的に伝わらない

## 参加フロー（現状）

```
スペースURL を開く
→ 「〜〜へようこそ！」メッセージ
→ ゲスト参加 or ログイン（or 新規登録）を選択
→ （ゲストを選択した場合）ニックネームを入力
→ 参加
```

## 修正内容

### 1. スペース名の表示を整理する

- 上部のスペース名は **グレーのタイトル1つのみ** に統一する（現在2つ表示されているうちグレーのものを残す）
- リンクコピーボタンは、**右上に小さく配置** する（タイトルと並列で大きく置かない）

**変更前**
```
[スペース名]
[スペース名 + リンクコピーボタン]
```

**変更後**
```
[スペース名（グレー）]         [🔗]  ← 右上に小さく
```

### 2. 投稿の視覚表現を変更する

- 投稿・気持ちのかけらを表す「小さな粒」を **星つぶ ⭐️** に変更する
- 星つぶをタップすると、**投稿者のニックネームと投稿内容** がポップアップ表示される

## 対象スコープ

- スマホ（モバイル）画面を主対象とする
- 既存の参加フロー自体は変更しない（UIの見た目のみ修正）
