## 概要

ログ一覧画面は 2 つの画面で構成される。

| 画面 | ルート | 説明 |
|---|---|---|
| **コメント一覧**（CommentsScreen） | `#comments` | アクティブなスペースの全投稿を一覧表示する共有ビュー |
| **マイログ**（MyLogsScreen） | `#mylogs` | ログインユーザー自身の投稿履歴を一覧表示するプライベートビュー |

両画面ともスティッキーヘッダーにフィルター／並び替えコントロールを持ち、投稿カードのリストをスクロール表示する。

## 1. コメント一覧（CommentsScreen）

### 1-1. 画面構成

```
┌──────────────────────────────────────┐
│ [TopRightMenu]                       │  ← 右上メニュー（sticky）
│ コメント一覧                          │
│ みんなの声が流れてくるよ                │
│ N 件の投稿                            │
│ [新着順] [❤️ いいね順]                │  ← 並び替えバー
│ [✓ コメント] [✓ 気持ち]              │  ← フィルターバー（FilterBar）
│ [タグチップ...] [タグで絞り込む▼]     │  ← タグフィルター（タグがある場合のみ）
├──────────────────────────────────────┤
│ ┌──────────────────────────────────┐ │
│ │ 投稿者名                          │ │
│ │ 😂 / 🎙 本文テキスト             │ │
│ │ [画像サムネイル]                  │ │
│ │ #タグ  #タグ                      │ │
│ │ HH:MM          🤍 / ❤️ N        │ │
│ └──────────────────────────────────┘ │
│ ...（繰り返し）                       │
└──────────────────────────────────────┘
```

### 1-2. 並び替えバー

ヘッダーの `filterContainer` 先頭に配置。

| ボタン | 動作 | デフォルト |
|---|---|---|
| **新着順** | `createdAt` の降順でソート | ✅ |
| **❤️ いいね順** | `likeCount` の降順でソート（同数の場合は順不同） | — |

- アクティブなボタンは紫（`#7c3aed`）の塗りつぶしで強調表示される
- `sortOrder` state（`'newest' | 'likes'`）で管理
- ソートはフィルタ適用前に行い、フィルタ結果がソート順を維持する

### 1-3. フィルターバー（FilterBar）

投稿種別をチェックボックスで絞り込む。設定は `localStorage` にスペースごとに永続化される（キー: `hossii.filters.<spaceId>`）。

| チェックボックス | 対象投稿 |
|---|---|
| **コメント** | `origin` が `manual`（または未設定）かつ `message` が空でない投稿・画像投稿 |
| **気持ち** | `emotion` フィールドが設定されている投稿（manual / auto 問わず） |

- 両方が OFF の場合、コメント・気持ちのいずれにも該当しない投稿（空メッセージかつ emotion なし）は常に表示される
- スペース切り替え時にそのスペースの保存設定を自動ロードする

### 1-4. タグフィルター

スペースの `presetTags` ＋ 実際の投稿に付いているタグ（`tags` / `hashtags`）の和集合を候補として表示。
**タグが 1 件以上ある場合のみ** フィルター行が表示される。

| 操作 | 挙動 |
|---|---|
| 「タグで絞り込む」ボタン押下 | ドロップダウンが開き候補タグを選択できる |
| タグチップの ✕ 押下 | そのタグのフィルターを解除 |
| 複数タグ選択 | いずれかのタグを持つ投稿を表示（OR 条件） |

### 1-5. 投稿カード

| 要素 | 表示条件 | 内容 |
|---|---|---|
| 投稿者名 | `authorName` が存在する場合 | ニックネーム表示 |
| ログアイコン | 笑いログ（`autoType === 'laughter'`）または 音声ログ（`autoType === 'speech'` / `logType === 'speech'`） | 😂 / 🎙 |
| 本文 | 笑いログ以外かつテキストがある場合 | テキスト（最大 3 行、スマホは 2 行） |
| 画像サムネイル | `imageUrl` あり ＋ `comments_thumbnail` フラグ ON | タップでライトボックス表示 |
| 画像リンク | `imageUrl` あり ＋ `comments_thumbnail` フラグ OFF | 「📎 画像を開く」外部リンク |
| タグ | `tags` または `hashtags` が存在する場合 | プリセットタグ（紫）・自由入力タグ（グレー） |
| タイムスタンプ | 常時 | `HH:MM` 形式 |
| いいねボタン | `likes_enabled` フラグ ON のスペースのみ | 🤍 / ❤️ + 件数 |

### 1-6. いいね機能

- `likes_enabled` フィーチャーフラグが ON のスペースでのみいいねボタンを表示
- ログイン済みユーザーの場合、画面ロード時に `fetchLikedIds` で自分がいいね済みの ID セットを取得
- ボタン押下時に楽観的更新（即時 UI 反映）し、`toggleLike` で Supabase を非同期更新
- 詳細仕様は [18_function_投稿へのいいね、コメント機能.md](./18_function_投稿へのいいね、コメント機能.md) を参照

### 1-7. 画像ライトボックス

- 画像サムネイルをタップすると全画面のライトボックスで拡大表示
- 背景クリックまたは ✕ ボタン、`Escape` キーで閉じる

## 2. マイログ（MyLogsScreen）

### 2-1. 画面構成

```
┌──────────────────────────────────────┐
│ [TopRightMenu]                       │
│ マイログ            [すべて][このスペース]│  ← スペースフィルター（右上）
│ あなたが残したログ                     │
│ [新着順] [❤️ いいね順]                │  ← 並び替えバー
├──────────────────────────────────────┤
│ N 件                                 │
│ ┌──────────────────────────────────┐ │
│ │ [スペース名ピル]          相対時刻 │ │
│ │ 本文テキスト（最大3行）           │ │
│ │ [画像サムネイル]                  │ │
│ │ 投稿者名               😊        │ │
│ └──────────────────────────────────┘ │
│ ...（繰り返し）                       │
└──────────────────────────────────────┘
```

### 2-2. スペースフィルター

| ボタン | 挙動 |
|---|---|
| **すべて** | 全スペースの自分の投稿を表示（デフォルト） |
| **このスペース** | `activeSpaceId` と一致する投稿のみ表示 |

- `profile.id` が未設定の場合（未ログイン）は結果が 0 件になる

### 2-3. 並び替えバー

コメント一覧と同仕様。

| ボタン | 動作 | デフォルト |
|---|---|---|
| **新着順** | `createdAt` の降順 | ✅ |
| **❤️ いいね順** | `likeCount` の降順 | — |

### 2-4. 投稿カード

| 要素 | 内容 |
|---|---|
| スペース名ピル | 投稿が属するスペース名（グラデーションのバッジ） |
| 相対時刻 | 「今」「N分前」「N時間前」または「M月D日」形式 |
| 本文 | テキスト（最大 3 行、スマホは 2 行で省略） |
| 画像サムネイル | `imageUrl` がある場合に表示。タップでライトボックス表示 |
| 投稿者名 | `authorName` がある場合に表示 |
| 気持ち絵文字 | `emotion` が設定されている場合に表示（`EMOJI_BY_EMOTION` で絵文字に変換） |

### 2-5. 画像ライトボックス

コメント一覧と同仕様（背景クリック・✕ ボタン・`Escape` で閉じる）。

### 2-6. 空の状態

| 条件 | 表示テキスト |
|---|---|
| `profile.id` が未設定 | まだ投稿がありません |
| スペースフィルター「このスペース」で該当なし | このスペースへの投稿はまだありません |
| 「すべて」で投稿なし | 投稿履歴がありません |

## 3. 共通仕様

### データ取得

| 画面 | 取得元 |
|---|---|
| コメント一覧 | `getActiveSpaceHossiis()`（アクティブスペースの全投稿） |
| マイログ | `state.hossiis`（全スペース分）から `authorId === profile.id` で絞り込み |

### ソートロジック（共通）

```
sortOrder === 'likes'
  ? (b.likeCount ?? 0) - (a.likeCount ?? 0)   // いいね数降順
  : b.createdAt.getTime() - a.createdAt.getTime()  // 新着降順
```

`likeCount` が未設定（`undefined`）の場合は `0` として扱う。

### レスポンシブ対応

- モバイル（`max-width: 768px`）: コンテナ下部に `padding-bottom: 5rem`（BottomNav 分の余白）
- スマホ（`max-width: 480px`）: カード本文の最大表示行数を 2 行に短縮
