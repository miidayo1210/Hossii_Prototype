## 概要

ログ一覧画面は 2 つの画面で構成される。

| 画面 | ルート | 説明 |
|---|---|---|
| **コメント一覧**（CommentsScreen） | `#comments` | アクティブなスペースの投稿一覧。**PaneFilterSegment** で「このタブ／すべてのタブ／各タブ」を切替（[93 §16](./93_Hossiiタブ機能追加.md) Phase 9B）。初期は現在選択中 Pane。**ヘッダーで「全体のログ／私のログ」を切替**（[83](./83_ログ一覧の私のログ切替.md)） |
| **マイログ**（MyLogsScreen） | `#mylogs` | **全スペース横断**の自分の投稿履歴（ナビからは未到達。直リンクのみ）。「このスペース」かつ Pane 2 件以上で Pane フィルタ（[93 Phase 10E](./93_Hossiiタブ機能追加.md)） |

`#comments` の「私のログ」は **現在スペース内**の `authorId === profile.id` の投稿のみ。`#mylogs` は全スペース横断。

両画面ともスティッキーヘッダーにフィルター／並び替えコントロールを持ち、投稿カードのリストをスクロール表示する。

## 1. コメント一覧（CommentsScreen）

### 1-1. 画面構成

```
┌──────────────────────────────────────┐
│ [TopRightMenu]                       │  ← 右上メニュー（sticky）
│ [全体のログ (N) │ 私のログ (M)]      │  ← スライドセグメント（[83](./83_ログ一覧の私のログ切替.md)）
│ [このタブ │ すべてのタブ │ ▼各タブ]   │  ← Pane 2件以上時のみ（[93 §16](./93_Hossiiタブ機能追加.md)）
│ コメント一覧 / 私のログ               │  ← スコープでタイトル切替
│ みんなの声が流れてくるよ / あなたが…   │
│ [🙂 ニックネーム として参加中]         │  ← 「私のログ」時のみ
│ N 件の投稿 / N 件                     │
│ [新着順] [❤️ いいね順]                │  ← 並び替えバー
│ [✓ コメント] [✓ 気持ち]              │  ← フィルターバー（FilterBar）
│ [タグチップ...] [タグで絞り込む▼]     │  ← タグフィルター（タグがある場合のみ）
├──────────────────────────────────────┤
│ ┌──────────────────────────────────┐ │
│ │ 投稿者名（全体のみ）              │ │
│ │ 😂 / 🎙 本文テキスト             │ │
│ │ [画像サムネイル]                  │ │
│ │ #タグ  #タグ                      │ │
│ │ HH:MM / 相対時刻   🤍 / ❤️ N     │  ← 私のログは相対時刻
│ └──────────────────────────────────┘ │
│ ...（繰り返し）                       │
└──────────────────────────────────────┘
```

### 1-1a. ログスコープ切替（全体／私）

詳細は [83_ログ一覧の私のログ切替](./83_ログ一覧の私のログ切替.md)。実装は `LogListBody` + `LogScopeSegment`。

| 項目 | 内容 |
|------|------|
| UI | ヘッダー最上段のスライドセグメント（左=全体、右=私）。件数バッジ付き |
| 永続化 | `localStorage` キー `hossii.logScope`（`all` / `mine`、デフォルト `all`） |
| 私のログ | `authorId === profile.id` の投稿のみ（現在スペース内） |
| クイックログ | `SpaceScreen` のパネルでも同一 UI（短縮ラベル「全体」「私」） |

### 1-1b. Pane フィルタ（Phase 9B）

Pane が 2 件以上のスペースでのみ、`LogScopeSegment` 直下に `PaneFilterSegment` を表示する。

| モード | 説明 | fetch query key |
|--------|------|-----------------|
| このタブ | タブバー / URL の active Pane に追従 | `pane:{activePaneId}:all:v2` |
| すべてのタブ | Space 内全 Pane 横断 | `pane:*:all:v2` |
| 各タブ（dropdown） | 指定 Pane のみ（URL は変えない） | `pane:{paneId}:all:v2` |

永続化: `localStorage` キー `hossii.commentsPaneFilter.{spaceId}`。クイックログパネルには未適用。

### 1-1c. 投稿の Pane 移動（Phase 10C）

管理者かつ Pane 2 件以上のスペースで、各投稿カードのメタ行に「タブへ」dropdown を表示する（「非表示」ボタンの左）。

| 項目 | 内容 |
|------|------|
| UI | `MoveHossiiPaneSelect` — 現在 Pane 以外の visible Pane を選択 |
| 権限 | 管理者のみ（`isAdmin`） |
| 動作 | 楽観更新 → `updateHossiiPaneId` → 失敗時ロールバック |
| 座標 | 移動時に `positionX/Y` は変更しない |

### 1-2. 並び替えバー

ヘッダーの `filterContainer` 先頭に配置。

| ボタン | 動作 | デフォルト |
|---|---|---|
| **新着順** | `createdAt` の降順でソート | ✅ |
| **❤️ いいね順** | `likeCount` の降順でソート（同数の場合は順不同） | — |

- アクティブなボタンは紫（`#7c3aed`）の塗りつぶしで強調表示される
- `sortOrder` state（`'newest' | 'likes'`）で管理
- ソートはフィルタ適用前に行い、フィルタ結果がソート順を維持する

### 1-3. フィルターバー（FilterBar）

投稿種別をチェックボックスで絞り込む。設定は `localStorage` にスペースごとに永続化される（キー: `hossii.filters.<spaceId>`）。

| チェックボックス | 対象投稿 |
|---|---|
| **コメント** | `origin` が `manual`（または未設定）かつ `message` が空でない投稿・画像投稿 |
| **気持ち** | `emotion` フィールドが設定されている投稿（manual / auto 問わず） |

- 両方が OFF の場合、コメント・気持ちのいずれにも該当しない投稿（空メッセージかつ emotion なし）は常に表示される
- スペース切り替え時にそのスペースの保存設定を自動ロードする

### 1-4. タグフィルター

スペースの `presetTags` ＋ 実際の投稿に付いているタグ（`tags` / `hashtags`）の和集合を候補として表示。
**タグが 1 件以上ある場合のみ** フィルター行が表示される。

| 操作 | 挙動 |
|---|---|
| 「タグで絞り込む」ボタン押下 | ドロップダウンが開き候補タグを選択できる |
| タグチップの ✕ 押下 | そのタグのフィルターを解除 |
| 複数タグ選択 | いずれかのタグを持つ投稿を表示（OR 条件） |

### 1-5. 投稿カード

| 要素 | 表示条件 | 内容 |
|---|---|---|
| 投稿者名 | `authorName` が存在する場合 | ニックネーム表示 |
| ログアイコン | 笑いログ（`autoType === 'laughter'`）または 音声ログ（`autoType === 'speech'` / `logType === 'speech'`） | 😂 / 🎙 |
| 本文 | 笑いログ以外かつテキストがある場合 | テキスト（最大 3 行、スマホは 2 行） |
| 画像サムネイル | `imageUrl` あり ＋ `comments_thumbnail` フラグ ON | タップでライトボックス表示 |
| 画像リンク | `imageUrl` あり ＋ `comments_thumbnail` フラグ OFF | 「📎 画像を開く」外部リンク |
| タグ | `tags` または `hashtags` が存在する場合 | プリセットタグ（紫）・自由入力タグ（グレー） |
| タイムスタンプ | 常時 | `HH:MM` 形式 |
| いいねボタン | `SpaceSettings.features.likesEnabled` が ON のスペースのみ | 🤍 / ❤️ + 件数 |

### 1-6. いいね機能

- `features.likesEnabled`（管理画面「投稿・交流ルール」）が ON のスペースでのみいいねボタンを表示
- ログイン済みユーザーの場合、画面ロード時に `fetchLikedIds` で自分がいいね済みの ID セットを取得
- ボタン押下時に楽観的更新（即時 UI 反映）し、`toggleLike` で Supabase を非同期更新
- 詳細仕様は [18_function_投稿へのいいね、コメント機能.md](./18_function_投稿へのいいね、コメント機能.md) を参照

### 1-7. 画像ライトボックス

- 画像サムネイルをタップすると全画面のライトボックスで拡大表示
- 背景クリックまたは ✕ ボタン、`Escape` キーで閉じる

## 2. マイログ（MyLogsScreen）

### 2-1. 画面構成

```
┌──────────────────────────────────────┐
│ [TopRightMenu]                       │
│ マイログ            [すべて][このスペース]│  ← スペースフィルター（右上）
│ あなたが残したログ                     │
│ [このタブ │ すべてのタブ │ ▼各タブ]   │  ← 「このスペース」+ Pane 2+ のみ
│ [新着順] [❤️ いいね順]                │  ← 並び替えバー
├──────────────────────────────────────┤
│ N 件                                 │
│ ┌──────────────────────────────────┐ │
│ │ [スペース名ピル]          相対時刻 │ │
│ │ 本文テキスト（最大3行）           │ │
│ │ [画像サムネイル]                  │ │
│ │ 投稿者名               😊        │ │
│ └──────────────────────────────────┘ │
│ ...（繰り返し）                       │
└──────────────────────────────────────┘
```

### 2-2. スペースフィルター

| ボタン | 挙動 |
|---|---|
| **すべて** | 全スペースの自分の投稿を表示（デフォルト） |
| **このスペース** | `activeSpaceId` と一致する投稿のみ表示 |

- `profile.id` が未設定の場合（未ログイン）は結果が 0 件になる

### 2-2b. Pane フィルタ（Phase 10E）

「このスペース」選択中かつ Pane が 2 件以上のとき、スペースフィルター直下に `PaneFilterSegment` を表示する（`#comments` Phase 9B と同 UI）。

| モード | 説明 |
|--------|------|
| このタブ | タブバーの active Pane に追従 |
| すべてのタブ | 現在スペース内の全 Pane 横断 |
| 各タブ（dropdown） | 指定 Pane のみ |

永続化: `localStorage` キー `hossii.mylogsPaneFilter.{spaceId}`（Comments とは別キー）。追加 fetch なし — store 内 `matchesPane` で filter。

「すべて」スペースフィルター選択時は Pane フィルタ非表示。

### 2-3. 並び替えバー

コメント一覧と同仕様。

| ボタン | 動作 | デフォルト |
|---|---|---|
| **新着順** | `createdAt` の降順 | ✅ |
| **❤️ いいね順** | `likeCount` の降順 | — |

### 2-4. 投稿カード

| 要素 | 内容 |
|---|---|
| スペース名ピル | 投稿が属するスペース名（グラデーションのバッジ） |
| 相対時刻 | 「今」「N分前」「N時間前」または「M月D日」形式 |
| 本文 | テキスト（最大 3 行、スマホは 2 行で省略） |
| 画像サムネイル | `imageUrl` がある場合に表示。タップでライトボックス表示 |
| 投稿者名 | `authorName` がある場合に表示 |
| 気持ち絵文字 | `emotion` が設定されている場合に表示（`EMOJI_BY_EMOTION` で絵文字に変換） |

### 2-5. 画像ライトボックス

コメント一覧と同仕様（背景クリック・✕ ボタン・`Escape` で閉じる）。

### 2-6. 空の状態

| 条件 | 表示テキスト |
|---|---|
| `profile.id` が未設定 | まだ投稿がありません |
| スペースフィルター「このスペース」で該当なし | このスペースへの投稿はまだありません |
| 「すべて」で投稿なし | 投稿履歴がありません |

## 3. 共通仕様

### データ取得

| 画面 | 取得元 |
|---|---|
| コメント一覧 | `getActiveSpaceHossiis()` / `getHossiisForQueryKey()`（Pane スコープは Phase 9B フィルタに従う） |
| マイログ | `state.hossiis`（全スペース分）から `authorId === profile.id` で絞り込み。Pane フィルタは store 内 filter |

### ソートロジック（共通）

```
sortOrder === 'likes'
  ? (b.likeCount ?? 0) - (a.likeCount ?? 0)   // いいね数降順
  : b.createdAt.getTime() - a.createdAt.getTime()  // 新着降順
```

`likeCount` が未設定（`undefined`）の場合は `0` として扱う。

### レスポンシブ対応

- モバイル（`max-width: 768px`）: コンテナ下部に `padding-bottom: 5rem`（BottomNav 分の余白）
- スマホ（`max-width: 480px`）: カード本文の最大表示行数を 2 行に短縮
