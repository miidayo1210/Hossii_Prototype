# 61 スペース表示中投稿数表示

> 最終更新: 2026-04-06

## 機能概要

スペース画面の左サイドバー（`LeftControlBar`）に、**現在の表示条件で画面上に並んでいる投稿数**をスペース右上へ出すトグルを置く。

期間（1日／1週／1月／全期）や表示件数（50／100／150／無制限）、表示モード（フル／バブル／画像のみ）を変えたとき、その条件に合わせた件数が一目で分かる。

---

## できること（実装済み）

- **PC・モバイル**で、左バーの **#（Hash）** ボタンを押すと、スペース**右上**に **「N 件」** のバッジが表示される。もう一度押すと非表示。
- 表示する数 **N** は、スペース上に実際に描画している投稿リスト（`displayHossiis`）の件数と一致する。
  - **期間フィルタ**（例: 1ヶ月）を適用したうえでの件数。
  - **表示件数上限**（例: 100）を適用したうえでの件数（実際に画面にある個数。100件未満ならその数）。
  - **画像のみモード**では、画像のない投稿は除外されたうえでの件数。
  - **非表示投稿**（`isHidden`）は件数に含めない。
- トグルのオン／オフは **`localStorage`（キー `hossii.showPostCountBadge`）** に保存され、リロード後も維持される。
- **投稿順モード**で右上に「降順／昇順」ツールバーが出ているときは、その**上**にバッジを縦に並べ、重なりを避ける。

## UI 仕様

### 左サイドバー（`LeftControlBar`）

| 項目 | 内容 |
|------|------|
| 位置 | 表示サイズ（Zoom）ボタンの直下、QR ボタンの直前 |
| アイコン | `#`（Lucide `Hash`） |
| 状態 | オンのとき既存のアクティブスタイル（グラデーション） |
| モバイル | `mobileVisible` により、モバイル左バーでも表示（期間・件数グループと同様） |

**表示条件:** 表示モードがスライドショー（`viewMode === 'slideshow'`）のときは `LeftControlBar` ごと非表示のため、このトグルも表示されない。

### スペース右上バッジ（`SpaceScreen`）

| 項目 | 内容 |
|------|------|
| 文言 | 数値（タブラー数字）＋「件」 |
| スタイル | 半透明背景・ぼかし・白系ボーダー（タイトルバッジや投稿順ツールバーと調和） |
| 配置 | `position: fixed`、PC では TopBar 直下付近（`top: calc(4rem + 10px)` 付近）、`z-index: 95` 前後 |

---

## 動作仕様（カウントの定義）

カウントは **フィルタ後・上限適用後の「いま画面にいる投稿」** の件数である。内部的には `displayHossiis.length` と同じ。

処理の流れ（概要）:

1. 非表示投稿を除く  
2. 期間の下限日時より古い投稿を除く（「全期」は期間フィルタなし）  
3. 「画像のみ」のときは画像 URL がない投稿を除く  
4. 作成日時の新しい順にソート  
5. 表示件数上限で `slice`（無制限は上限なし）  

スライドショーモードに切り替えたあとも、内部の一覧は同じ `displayHossiis` を使うため、先にトグルをオンにしておけば、スライドショー中も同じ件数が右上に残る（左バーは非表示のまま）。
