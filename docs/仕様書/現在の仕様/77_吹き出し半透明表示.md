# 77 吹き出し半透明表示

> **分類:** `[UI/UX]` スペース上の吹き出し（Bubble）の背景透過  
> **関連:** [30_スペースの吹き出し](./30_スペースの吹き出し.md) / [54_吹き出しの色カラーパレット](./54_吹き出しの色カラーパレット%20.md) / [70_背景画像サイズ](./70_背景画像サイズ.md)  
> **ステータス:** 仕様確定・実装済み（`bubbleColorAlpha.ts`・`Tree.tsx`・`SpaceScreen.module.css`）

> 最終更新: 2026-06-02（CSS 変数方式・hover/active 統一）

---

## 1. 目的

スペースの **背景（壁紙・パターン）** が、吹き出しの背後から **なんとなく見える** ようにする。  
投稿の可読性は保ちつつ、スペース全体の一体感を損なわない **半透明ガラス風** を標準とする。

---

## 2. 適用対象と例外

| 種類 | 半透明 | 備考 |
|------|--------|------|
| 通常吹き出し（テキスト・画像） | ✅ alpha **0.80** + `backdrop-filter: blur(4px)` | デフォルト |
| カスタム色（`bubbleColor`） | ✅ 表示時に `withBubbleAlpha()` で **0.80** | hex のまま DB 保存 |
| PNG 形状吹き出し | ✅ フィル色 alpha **0.80** + blur | 形状 PNG は上にオーバーレイ |
| hover / active | ✅ alpha **0.90** | CSS 変数 `--bubble-bg-hover` |
| キャンバス投稿（`bubbleCanvas`） | ❌ **不透明** | 画像カード本体のため対象外 |
| モバイル StarView（星） | ❌ 対象外 | 点表示のみ |

---

## 3. 旧ベースライン（参考）

| 箇所 | 現状 |
|------|------|
| デフォルト CSS（`.bubble`） | `rgba(255, 251, 235, 0.85)` + `backdrop-filter: blur(4px)` — **既に半透明** |
| hover | `0.95` にやや不透明化 |
| ユーザー選択色（`bubbleColor`） | **不透明 hex** をそのまま `backgroundColor` に適用（[Tree.tsx](../../../src/components/SpaceScreen/Tree.tsx)）→ 壁紙が透けない |
| PNG 形状吹き出し | 色未指定時 `rgba(..., 0.95)`、`backdrop-filter: none` |

**課題:** カスタム色・形状付き吹き出しだけ不透明になり、体験がばらつく。

---

## 4. 仕様（確定方針）

### 4.1 通常吹き出し（テキスト・画像）

| 項目 | 値 |
|------|-----|
| 背景 alpha | **0.72 〜 0.85**（既定 **0.80**） |
| ぼかし | `backdrop-filter: blur(4px)` を維持 |
| 枠線 | `--bubble-border`（背景色と同系、alpha 0.90 相当） |
| hover / active | alpha **0.90**（`--bubble-bg-hover`） |

### 4.2 CSS 変数方式（実装）

`Tree.tsx` が吹き出しごとに CSS 変数を設定し、CSS の hover / active でも同じ色が効く。

```tsx
bubbleStyle['--bubble-bg'] = withBubbleAlpha(fillColor, 0.80);
bubbleStyle['--bubble-bg-hover'] = withBubbleAlpha(fillColor, 0.90);
```

```css
.bubble {
  background: var(--bubble-bg, rgba(255, 251, 235, 0.80));
}
.bubble:hover,
.bubbleActive {
  background: var(--bubble-bg-hover, rgba(255, 251, 235, 0.90));
}
```

### 4.3 ユーザー選択色（`bubbleColor`）

- 保存値は **hex のまま**（DB・投稿画面の互換維持）
- **表示時**に alpha を付与して `rgba(r,g,b,0.80)` 相当に変換する
- 既に `rgba(...)` 形式の値は alpha だけ仕様範囲にクランプ

### 4.4 PNG 形状吹き出し

- フィル色にも **同じ alpha 方針** を適用（形状 PNG の下に壁紙がうっすら見える）
- `backdrop-filter: blur(4px)` を維持（以前は `none` だったが半透明優先で復活）

### 4.5 キャンバス投稿（`bubbleCanvas`）

- 画像カード本体は **不透明のまま**（変更しない）
- 外枠シャドウのみ現状維持

### 4.6 モバイル StarView

- 星そのものは点表示のため **対象外**
- 星の **プレビューバブル**（`.previewBubble`）は別仕様（[75](./75_スマホ横持ちレイアウト.md)）
