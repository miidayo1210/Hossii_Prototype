---

## 1. 目的

アイコンのみのボタンが多い左コントロールバー・右上メニュー等で、  
**ホバー（デスクトップ）やロングプレス（タッチ）** の際に短い説明を表示し、  
初見ユーザーが機能を把握しやすくする。

---

## 2. 共通仕様

| 項目 | 値 |
|------|----|
| 表示ディレイ | **400ms**（素早いカーソル移動では出ない） |
| 非表示ディレイ | **100ms**（すぐに消す） |
| アニメーション | `opacity 0 → 1` + `translateX(-4px) → translateX(0)` を **0.15s ease-out** |
| 最大幅 | **220px**（2 行まで。超過は `...` 省略） |
| テキスト | 1〜2 行・簡潔な日本語 |
| ポインタイベント | `pointer-events: none`（ツールチップ自体はクリック不可） |
| `title` 属性 | ブラウザネイティブ tooltip と二重表示しないよう、tooltip 表示中は `title` を `""` に差し替えるか、JS 側でネイティブ抑止 |

## 4. 位置ルール

| 元コントロール | ツールチップの方向 |
|---------------|-----------------|
| 左コントロールバー（縦並び）| **右へ** (`left: calc(100% + 8px)`) |
| 左コントロールバー（横並び・スマホ縦）| **下へ** (`top: calc(100% + 6px)`) |
| 下部ナビバー（BottomNavBar）| **上へ** (`bottom: calc(100% + 6px)`) |
| 右上メニュー | **下へ** (`top: calc(100% + 6px)`) |

## 5. 各ボタンのツールチップ内容

### 左コントロールバー（主要ボタン）

| ボタン（アイコン） | ON 状態テキスト | OFF 状態テキスト |
|-------------------|----------------|----------------|
| Maximize2 / Minimize2 | スペースを全画面表示中。クリックで戻す | スペースを画面いっぱいに広げる |
| Eye / EyeOff | Hossii キャラを表示中 | Hossii キャラを非表示中 |
| Mic / MicOff | 音声認識 ON。話しかけると投稿できます | 音声認識 OFF |
| Volume2 / VolumeX | Hossii の読み上げ ON（タップで OFF） | Hossii の読み上げをオンにする |
| 🏝（島） | 隣のスペースにワープ | 自スペースに戻る |

### `...`（オーバーフロー）展開後のボタン

| ボタン | テキスト |
|--------|---------|
| ZoomIn + % | バブルの表示サイズを変更（クリックで切替） |
| Hash (#) | 現在の表示件数バッジを ON/OFF |
| QrCode | QR コードパネルを表示/非表示 |
| 📝 フル | フルサイズで全投稿を表示 |
| 💬 バブル | テキストのみ・コンパクト表示 |
| 🖼 画像 | 画像付き投稿のみ表示 |
| 🎞 スライドショー | 画像を順番に全画面表示 |
| 🔀 ランダム | 投稿をランダムに配置 |
| ↔️ 投稿順 | 投稿を時系列の格子に整列 |
| 期間ボタン（1日 等） | 過去〇の投稿を表示 |
| 件数ボタン（50 等） | 最大〇件を表示 |

## 6. 実装方針

### 実装メモ

- コンポーネント: `src/components/common/Tooltip.tsx`
- ラッパーは **`display: contents`**。モバイル左バーでは `.controlButton:not(.mobileVisible)` で PC 専用ボタンのみ非表示にする（[33](./33_スペース左サイドのバー.md) §4-1）。

### アクセシビリティ

- `role="tooltip"` + `id` をツールチップに付与し、ボタンの `aria-describedby` で紐づける
- `prefers-reduced-motion: reduce` の場合はアニメーションなし（即表示/消去）
