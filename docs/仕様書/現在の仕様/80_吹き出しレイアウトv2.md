## 1. 課題と目標

| 課題 | 目標 |
|------|------|
| 投稿が増えるとごちゃごちゃして読めない | ひとつひとつをコンパクトに整え、密集しても識別しやすくする |
| メタ情報（日時・タグ）がコメント本文と同等の大きさで並ぶ | メタ情報を小さく・まとめて1行に収め、本文を際立たせる |
| いいねボタンがフッターを占有して吹き出しを縦に伸ばす | いいねを吹き出し**外側**の小さな浮きバッジにし、本体サイズを変えない |
| カスタム色が不透明で壁紙に馴染まない | 全吹き出しをガラス風半透明（alpha 0.80）で統一（[77](./77_吹き出し半透明表示.md)） |

## 2. 新レイアウト概要

```
┌─────────────────────────────────┐
│ 😊  ニックネーム                 │  ← ヘッダー行（絵文字 + 名前）
│                                 │
│  コメント本文                    │  ← メイン（最大 3 行・優先表示）
│  [画像サムネイル（あれば）]       │
│                                 │
│  5分前  #タグA #タグB           │  ← フッター行（小さい・1行）
└─────────────────────────────────┘
                              ❤️ 3  ← フローティングいいねバッジ（外側右下）
```

---

## 3. ヘッダー行

| 要素 | 仕様 |
|------|------|
| 絵文字 | `font-size: 1.25rem`（20px）。左端 |
| ニックネーム | `font-size: 0.75rem`（12px）、`font-weight: 600`。絵文字の右に隣接 |
| 行全体の高さ | `min-height: 1.75rem`（28px） |
| 区切り | ヘッダー行とコメント本文の間に `margin-bottom: 0.375rem` |

---

## 4. コメント本文エリア

| 要素 | 仕様 |
|------|------|
| フォントサイズ | `0.8125rem`（13px）。現行と同等 |
| 最大行数 | `viewMode === 'full'`: 最大 **3 行**（`-webkit-line-clamp: 3`）。`viewMode === 'bubble'`: 最大 **2 行** |
| overflow | 超過分は `...` 省略 |
| padding | `0`（ヘッダー行の下余白で十分） |

---

## 5. フッター行（メタ情報コンパクトライン）

フッター行は **1 行**で収める。ボーダー区切り線は設けず、余白のみで本文と分離。

```
5分前  ·  #タグA  #タグB  +2
```

| 要素 | 仕様 |
|------|------|
| 時刻アイコン | **なし**（テキストのみ。絵文字・Clock アイコンは使わない） |
| 日時 | `font-size: 0.625rem`（10px）、`opacity: 0.6`。「X分前 / X時間前 / X日前」形式 |
| 区切り | `·`（中点、`opacity: 0.4`） |
| タグ | `font-size: 0.625rem`（10px）、`opacity: 0.65`。最大 **2 件**表示。超過は `+N` |
| 行全体の高さ | `1.25rem`（20px）固定 |
| `viewMode === 'bubble'` | フッター行を **非表示** にしてさらにコンパクトに |
| タグ 0 件かつ日時なし | フッター行自体を非表示 |

## 6. フローティングいいねバッジ

いいねボタンを **吹き出しの外側・右下に飛び出す小さなバッジ** として配置。
吹き出し本体のサイズ（width / height）は変動しない。

### 6-1. 配置

```css
/* 吹き出し div に position: relative が必要 */
.likeFloatingBadge {
  position: absolute;
  bottom: -0.875rem;  /* バッジの半分が外に出る */
  right: -0.5rem;
  z-index: 5;
}
```

### 6-2. バッジのスタイル

```
❤️ 3
```

| 項目 | 値 |
|------|----|
| 背景 | `rgba(255, 255, 255, 0.92)` + `backdrop-filter: blur(6px)` |
| ボーダー | `1px solid rgba(239, 68, 68, 0.25)` |
| border-radius | `1rem`（pill 形状） |
| padding | `0.125rem 0.375rem` |
| font-size | `0.6875rem`（11px） |
| アイコン | `❤️` または `♡`（未いいね時は灰色） |
| カウント | `0` のときは **非表示**（吹き出し hover 時のみゴースト `♡` 表示・案A）。`1+` は常時表示 |
| ホバー（0件時） | 吹き出し hover で opacity 0.55 のゴーストバッジが出現。クリックでいいね可能 |
| ホバー（1件以上） | バッジ `scale(1.15)` |
| クリック | 既存の bounce + particle アニメーションを継続（[31](./31_いいねボタンUI.md)） |

### 6-3. `likesEnabled` が OFF の場合

バッジを **描画しない**（`overflow: visible` の外に何も出さない）。

### 6-4. 吹き出しの `overflow` 対応

バッジが外にはみ出るため、吹き出し div の `overflow: hidden` を **廃止**して `overflow: visible` にする。  
ただし角丸・shadow はそのまま維持。  
吹き出しが重なる場合は `z-index` でバッジが他バブルの下に潜り込まないよう、吹き出し hover 時に `z-index` を上げる（既存の挙動を踏襲）。

## 8. ガラス風半透明（[77](./77_吹き出し半透明表示.md) との統合）

| 項目 | 値 |
|------|-----|
| 背景 alpha（デフォルト） | **0.80**（`rgba(255, 251, 235, 0.80)`） |
| backdrop-filter | `blur(4px)` |
| カスタム色 | `withBubbleAlpha(bubbleColor)` で表示時に alpha 付与（[77](./77_吹き出し半透明表示.md)） |
| hover / active | alpha **0.90** |

## 9. `viewMode` 別の表示差分

| viewMode | ヘッダー | コメント | フッター行 | いいねバッジ |
|----------|---------|---------|-----------|------------|
| `full` | あり | 最大 3 行 | あり | あり |
| `bubble` | あり | 最大 2 行 | **なし** | あり（カウント > 0 のみ） |
| `image` | 画像ありのみ表示 | 最大 1 行 | あり | あり |
