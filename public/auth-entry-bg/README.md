# 認証エントリー背景画像（トップ・ログイン・管理者ログイン共通）

仕様: [65_トップページ.md](../../docs/仕様書/現在の仕様/65_トップページ.md) の「背景用静止画の要件」。

## 置き方

1. このフォルダに **WebP または JPEG** を配置する（例: `01.webp` … `06.webp`）。
2. [authEntryBg.config.ts](../../src/components/Auth/authEntryBg.config.ts) の `AUTH_ENTRY_BG_IMAGES` に **`/auth-entry-bg/ファイル名`** を列挙する。

## 目安

- **枚数**: 推奨 4〜6、最低 3
- **比率**: 16:9、**解像度**: 1920×1080 前後
- **1 枚あたり**: 約 200〜600 KB に圧縮
- **権利**: 自作・依頼・ライセンス済みのみ

画像が 0 枚（配列空）のときは、従来どおり CSS グラデのみ表示されます。
